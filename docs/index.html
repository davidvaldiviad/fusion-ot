Enhancing time-frequency resolution via optimal transport fusion of spectrograms

David Valdivia1   Elsa Cazelles1,2   Cédric Févotte1,2

1IRIT, Université de Toulouse     2CNRS

Non-stationary time signals have time-varying spectral properties that require tools such as the short-time Fourier transform to be properly understood. To compute the STFT, a signal of interest is segmented and multiplied with a sliding analysis window whose duration is fixed. While a larger window provides a better frequency resolution and a shorter one provides better temporal resolution, there is a limit to how well the STFT can properly locate energy in both dimensions. In this paper, we propose an optimal transport (OT) based method to overcome this problem by fusing the resolution of two input spectrograms.

In this website we share the tool we built to compute super-resolution spectrograms on audio signals. You can use one of the recordings we propose, use your microphone or upload a file you have.
